# IronCore Gym Management System

A responsive gym management web application developed to streamline day-to-day gym operations through an intuitive and user-friendly interface. The system enables gym members and administrators to efficiently manage memberships, trainer assignments, class scheduling, billing, and user profile management.

This project was developed as part of my Bachelor of Science (Hons) in Computer Science at University College Birmingham, demonstrating responsive web development, user interface design, and software engineering principles.

---

## Overview

IronCore is designed to simplify gym management by providing separate interfaces for members and administrators. The application focuses on improving user experience while supporting essential gym management operations in a clean and responsive environment.

## Getting Started

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in your preferred web browser.
4. Navigate through the application using the available pages.

No additional software or installation is required.
